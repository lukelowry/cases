# EuropeV2

European transmission grid built from a GridKit ENTSO-E extract. A separate case from [EuropeV1](../EuropeV1/).

- [EuropeV2.pruned.lat](EuropeV2.pruned.lat) and [EuropeV2.unpruned.lat](EuropeV2.unpruned.lat) are builds with and without network pruning.
- [EuropeV2.map.html](EuropeV2.map.html) is an interactive map view.
- [upstream/](upstream/) holds the source GridKit CSV tables.
- [tools/](tools/) holds the scripts that build the case from them.
