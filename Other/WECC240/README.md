# WECC240

Reduced 240-bus WECC model by NREL, modified by the IEEE-NASPI Oscillation Source Location Committee: 243 buses, 146 generating units at 56 plants, 329 lines, 122 transformers, and 139 loads, with 10 IEEEST PSS models added.

Reference: Yuan, Sen Biswas, Tan, Zhang, "Developing a Reduced 240-Bus WECC Dynamic Model for Frequency Response Study of High Renewable Integration," 2020 IEEE T&D.
